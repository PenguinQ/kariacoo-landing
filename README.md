# Dashtomer Landing
This is a redesign of current Dashtomer landing page.

## Development
### Requirements
- node ^22.3.0
- pnpm ^9.3.9

### Getting Started
1. Install dependencies by running:
   ```
   pnpm install
   ```

2. Build the component by running:
   ```
   pnpm run build
   
   or

   pnpm build
   ```

3. To preview, run:
   ```
   pnpm run dev
   
   or

   pnpm dev
   ```
